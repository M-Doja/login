A starter template

When building this template, one of my goals that I set out to fulfill was to create a template that could be used for multiple uses, as well as developing a reuseable CSS library as well. Built from a mobile first perspective, this responsive template can be altered slightly to create stunning web apps, sites that feature basic user authentication for an eCommerce store, or even a social media dashboard style platform. The options are limited only to your imagination.
